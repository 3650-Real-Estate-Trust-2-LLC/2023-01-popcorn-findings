# FUNCTIONS GUARANTEED TO REVERT WHEN CALLED BY NORMAL USERS CAN BE MARKED `PAYABLE`

##### A function with the modifier `onlyOwner` or `onlyX` will revert if a normal user tries to call it. Making the function payable will lower gas costs for legitimate callers as it skips the checks for payment, avoiding an average of 21 gas per call and deployment cost.
